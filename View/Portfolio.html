C++

Birdnest

Partial 3D OpenGL engine made using tutorial in https://learnopengl.com/ 2016 spring. Reads 3D model and text font styles information using libraries. Reads simple shaders which give models a texture and you can use it to give text new colors. Made own camera system that works quite well in 3D space, but I dont remember if I fixed the axel locks caused by euler angles. Took a little too big bite, but learned a lot about C++, OpenGL and class interfaces. Idea was to add enough methods to fly in 3D space with a bird model. Osittainen 3D OpenGL moottori tehty https://learnopengl.com/ tutoriaalin pohjalta. Lukee 3D malli tiedoston ja teksti fontit kirjastoja käyttäen. Lukee yksinkertaisia shadereitä joilla annetaan värit modeleille ja sillä voi määrittää tekstille uudet värit. Tein oman kamera systeemin joka toimi mielestäni hyvin 3D maailmassa, mutte en muista korjasinko siitä akselien lukkiutumisen euler kulmilla. Haukkasin vähän liian ison palan, mutta opin paljon C++, OpenGL ja luokka rakenteita.

Android

Moveometri

A 3 man group project for Mobile Project course. Application utilizes androids location and time and uses Google maps api to provide a trip timer. When you have made trips, by logging in your new trips can be saved in an external database! The server is coded in D language and uses Vibe.d library and it grants it REST API methods.

I made the timer activity, which includes coding for the times, locations and Google map API for the application. 3 hengen projekti Mobile Project kurssia varten. Ohjelma käyttää Androidin omaa location systeemiä ja aikaa. Locationit piirretään Google service Maps APIa käyttäen kartalle. Kun matkoja on tehty, ne voi siirtää ulkoiseen tietokantaan kirjautumisella. Tietokannan toiminnot on koodattu D kielellä ja käyttää Vibe.d kirjastoa saadakseen REST API toiminnot.

Tein projektin ajastus aktiviteetin, eli koodasin ajan, sijainnin ja Googlen kartan APIn ohjelmaan.
